# 單位經濟

本節描述 Aurora Protocol 單一批次的經濟結構——平台上融資操作的基本單元。理解批次層級的經濟模型，對評估投資機會的 DeFi 參與者及結構化融資申請的發起人均至關重要。

---

## 批次作為經濟單元

Aurora Protocol 上的每個批次均為獨立的融資操作，擁有自身的：

- 融資目標（以 USDC 計價）
- 代幣供應量與價格
- 里程碑計劃與釋放比例
- 費用結構
- 預期收益結構
- 獨立風險特徵

批次之間不存在組合層級的資金池、交叉補貼或共同責任。每個批次依其自身條件成功或失敗。

---

## 批次結構範例

以下範例說明典型農業融資批次的經濟結構：

### 批次參數

| 參數 | 數值 |
|---|---|
| 作物類型 | 泰國茉莉香米 |
| 融資金額 | 100,000 USDC |
| 代幣供應量 | 10,000 枚 RWA 代幣 |
| 代幣價格 | 10 USDC |
| 批次期限 | 6 個月（1 個作物週期） |
| 預期年化毛報酬率 | 12%（6 個月週期為 6%） |
| 平台費用 | 2.5%（基於本金，於里程碑釋放時扣除） |

### 資金流向

```
DeFi 參與者
       │
       │ 100,000 USDC
       ▼
   PrimarySale
       │
       │ 100,000 USDC
       ▼
   EscrowVault ──────── 平台費用：2,500 USDC ──────► Aurora Labs
       │
       │ 97,500 USDC（分三個里程碑淨額撥付予發起人）
       ▼
   發起人
       │
       │ 106,000 USDC（本金 + 收益）
       ▼
   ClaimVault
       │
       │ 106,000 USDC（透過銷毀領取分配）
       ▼
DeFi 參與者
```

### 參與者回報

| 指標 | 數值 |
|---|---|
| 每枚代幣投入 | 10 USDC |
| 每枚代幣回報（成功時） | 10.60 USDC |
| 每枚代幣淨收益 | 0.60 USDC（6 個月週期 6%） |
| 年化收益率（毛） | 約 12% |
| 平台費用承擔方 | 發起人（從撥付中扣除） |

### 發起人經濟

| 指標 | 數值 |
|---|---|
| 獲得融資（扣除費用後） | 97,500 USDC |
| 還款義務 | 106,000 USDC |
| 實際資金成本 | 約 8.7%（6 個月）/ 約 17.4%（年化） |
| 支付平台費用 | 2,500 USDC |

## 費用扣除機制

平台費用於每個里程碑資金釋放時按比例扣除。這意味著費用分散於整個批次生命週期，而非集中前置收取。

**三里程碑範例（30/35/35 分配）：**

| 里程碑 | 毛釋放額 | 費用（2.5%） | 發起人淨收額 |
|---|---|---|---|
| 里程碑 1 | 30,000 USDC | 750 USDC | 29,250 USDC |
| 里程碑 2 | 35,000 USDC | 875 USDC | 34,125 USDC |
| 里程碑 3 | 35,000 USDC | 875 USDC | 34,125 USDC |
| **合計** | **100,000 USDC** | **2,500 USDC** | **97,500 USDC** |

---

## 收益免責聲明

預估收益率為**基於發起人預期作物經濟的估算值**，並非保證。實際回報取決於：

- 作物表現及收穫品質
- 銷售時點的大宗商品市場價格
- 發起人的營運執行力
- 外部因素（天氣、物流、政策變動）
- 發起人的還款能力與意願

批次可能回報低於預期、僅返還本金，或導致部分乃至全部資金損失。詳見[風險揭露](../Risks.md)。

## 與傳統農業融資之比較

| 維度 | 傳統方式 | Aurora Protocol |
|---|---|---|
| 最低投資門檻 | 50,000 至 500,000 美元以上 | 低至 1 枚代幣（約 10 美元等值） |
| 地理限制 | 當地銀行、區域性金融機構 | 全球、無許可 |
| 透明度 | 季度報告，能見度有限 | 鏈上即時狀態 |
| 結算方式 | 30 至 90 天人工流程 | 智能合約確定性執行 |
| 中間人 | 多層（銀行、經紀商、代理人） | 參與者與發起人透過智能合約直接對接 |
| 流動性 | 非流動、固定期限承諾 | 目前非流動；二級市場規劃中 |
